Content‑based clustering using K‑Means over genres, rating, and runtime.

How it works

  1. We vectorize main and side genres.
  2. We scale numeric features (rating & runtime).
  3. We cluster with K‑Means (k=180), then recommend from the same cluster ranked by cosine similarity.
